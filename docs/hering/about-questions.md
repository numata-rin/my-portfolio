# About ページ改修 質問事項

## Q1. 「現在の活動」セクションの表示形式

`About-details.md` には文章形式とカード形式の両案が記載されています。
Homeページから削除したカード（個人開発・研究・学内カンパニー）をAboutに移すとのことでしたが、どちらの形式にしますか？

**候補A: カード形式**（Homeで削除したものと同じ構成）
```
個人開発       React / Next.js / Django を用いたWebアプリ開発
研究           情報推薦・行動支援・学習支援に関心
学内カンパニー  iFive でWebアプリ開発担当として活動
```

**候補B: 文章形式**（セクション内にまとめて文章で記述）

- **解答**
カード形式でお願いします。

最初に文章でHomeページと同じように活動部分の概要的なものを書いて、下に３つのカードを用意してほしいです。
３つのカードはそれぞれ縦に並べてください。
カードは角が丸いもので、サイズは現在表示されている文章部分と同じぐらいのサイズにしてほしいです。
３つのカードにはタイトルで「研究/study」みたいな感じで書いて、詳しい活動内容を中に書くみたいな感じでやってほしいです。
